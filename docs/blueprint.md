# **App Name**: FabricMind

## Core Features:

- AI-Powered Fabric Suggestions: Provide personalized fabric suggestions based on user queries using AI. This tool will reduce time spent on fabric selection and improve customer satisfaction.
- Visual Search: Allow users to upload images of fabrics or patterns they like and find similar options within the platform.
- Fabric Information Display: Showcase high-quality images, detailed descriptions, and available colors/patterns for each fabric.
- Virtual Visualization: Allow users to visualize selected fabrics in different room settings via uploaded or pre-loaded templates.
- User Query and Preference Storage: Store user queries and preferences to refine and personalize future recommendations.
- Vendor Integration and Purchase: Enable users to purchase fabrics directly through integrated vendor partners.
- Personalized Mood Boards: Allow users to create mood boards using selected fabrics and share them.

## Style Guidelines:

- Primary color: Deep teal (#008080) to convey sophistication and reliability, inspired by classic textile design.
- Background color: Light beige (#F5F5DC) to offer a neutral and calming backdrop.
- Accent color: Muted gold (#B8860B) for highlighting key elements and calls to action, referencing high-end decor.
- Body text font: 'PT Sans', sans-serif, suitable for the longer text found in fabric descriptions.
- Headline font: 'Playfair', serif, to add an elegant, high-end touch, particularly on shorter text elements.
- Use clear, representative icons to symbolize different fabric properties and categories.
- Implement subtle animations during the loading of AI suggestions to maintain user engagement.
